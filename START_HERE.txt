⚡ QUICK START (3 STEPS)
═══════════════════════════════════════════════════════════════════════

1️⃣  Setup Google Cloud Credentials
    • Go to console.cloud.google.com
    • Create project
    • Enable "Cloud Speech-to-Text API"
    • Create Service Account
    • Download JSON key
    • Save as: google-cloud-key.json (project root)

2️⃣  Install Dependencies
    npm install
    cp .env.example .env
    # Edit .env: GOOGLE_APPLICATION_CREDENTIALS=./google-cloud-key.json

3️⃣  Run Application
    npm start
    # Open http://localhost:3000 in browser
